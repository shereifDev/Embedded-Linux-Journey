## 📁 Repository Structure
Based on the course flow, this repository is organized into the following key directories:

* 📂 **`Summaries/`**: Contains comprehensive notes and summaries for the lectures (e.g., lecture summaries converted to `index.html`).
* 📂 **`projects/`**: Practical, hands-on projects built throughout the course to apply Embedded Linux concepts.
* 📂 **`tasks/`**: Regular assignments, exercises, and coding tasks to test and solidify your knowledge.
